PROJECT INFORMATION

The project titled “Ransomware Early Warning System” was developed by K.Gopika, V.Hema, SK.Nasrin, and M.Reshma as part of a Cyber Security Internship.

The main objective of this project is to detect and warn against ransomware attacks at an early stage by continuously monitoring file system activities, analyzing behavioral patterns, and generating real-time risk alerts before severe damage occurs.
